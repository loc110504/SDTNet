# Scribble-supervised Learning for Medical Image Segmentation

This repository provides re-implementations of several scribble-supervised methods for medical image segmentation:

1. [CycleMix – CVPR 2022](https://arxiv.org/abs/2203.01475)  
2. [DMPLS – MICCAI 2022](https://arxiv.org/abs/2203.02106)  
3. [ShapePU – MICCAI 2022](https://arxiv.org/pdf/2206.02118)
4. [UAMT - Pattern Recognition 2022](https://www.sciencedirect.com/science/article/pii/S0031320321005215)
5. [ScribFormer - TMI 2024](https://arxiv.org/pdf/2402.02029)
6. [DMSPS - MedIA 2024](https://www.sciencedirect.com/science/article/abs/pii/S1361841524001993?dgcid=author)
7. [TABNet - ArXiv 2025](https://arxiv.org/pdf/2507.02399)
